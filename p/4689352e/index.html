如何重新或全新安装 Windows 11？

重新或全新的意思就是安装前系统盘的所有数据都将被删除，然后安装新的系统。
如果你想保留现有系统盘中的数据，将系统升级到 Windows 11，请移步至 Windows 11 Installation Assistant。

随着微软制作了安装 Windows 的程序，那些不懂电脑的人不再需要像以前需要找电脑店或他人付费安装系统了。
使用 Windows 官方的安装程序只需要安装程序提示进行安装即可，安装过程很简单。

检测电脑能否安装 Windows 11

检测方法如下：

  • 查看 Windows 11 规格和系统要求 | Microsoft 。
  • 如果您的设备已经在运行 Windows 10，您可以使用电脑健康状况检查应用来评估兼容性。

重新或全新安装 Windows 11

  1. 准备一个不小于 8 GB 的 U 盘或 DVD。

  2. 打开 Create Windows 11 Installation Media。

  3. 下载 Create Windows 11 Installation Media。

  4. 运行下载后的 exe 文件，按照提示创建 Windows 11 安装介质。

  5. 创建完成后，重启电脑。重启时不停的按下 F12、F2、F11、ESC、F9键来调出启动项选择界面。选择你的 U 盘或 DVD。

  6. 后面的步骤就很简单了，下面我只列出一些注意步骤。

  7. 安装类型，重装的话那就是选第二个自定义咯。

  8. 下图请选择要安装的系统盘，别选到数据盘了。

    如果你还需要保留以前 Windows 安装中的文件，请直接点击 下一步，然后在弹出的界面中点击确定。

    我不需要保留系统盘中的文件，因此先点击 *格式化(F)*，格式化系统盘后再点击下一步。

  9. 一步一步，耐心安装即可。
