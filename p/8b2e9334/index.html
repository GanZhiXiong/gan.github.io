Microsoft Remote Desktop如何拷贝文件到Mac ？

你以为Microsoft Remote Desktop通过复制、粘贴快捷键就可以将远程桌面Windows中的文件拷贝到Mac中吗？
试试你就知道，这样虽然能将文件拷贝到Mac，但是却和源文件不同。

比如你在Windows上新建一个1.txt，并写入内容，选中文件，然后按下Ctrl + C，接着再Mac的Finder目录中按下Comand + V。你会发现文件虽然拷贝过来了，但是打开1.txt的内容却为空。

那么如何正确的将Windows中文件拷贝到Mac中呢？
正确步骤如下：

  1. 点击编辑PC按钮。

  2. 勾选 Redirect folders。

  3. 添加要将Windows文件拷贝到Mac中的目录。比如我配置为dell目录（如上图所示）。

  4. 重新连接Windows，你就会在文件资源管理器中发现Mac共享的文件夹dell。

  5. 将要拷贝到Mac中的文件直接拷贝到共享目录dell即可。
